# 数据流设计

## 消息发送数据流

```
User Input
    │
    ▼
ChatManager.sendMessage()
    │
    ├── MemoryManager.getRelevantMemories(context)
    │       └── StorageService.queryMemories()
    │
    ├── 构建 API 请求 (messages + system prompt + memories)
    │
    ▼
ClaudeApiClient.streamMessage()
    │
    ▼ (SSE chunks)
MessageModel.updateLast(chunk)
    │
    ▼
QML ListView (UI 更新)
    │
    ▼ (完成)
StorageService.saveMessage()
```

## 记忆提取数据流

```
对话完成
    │
    ▼
MemoryExtractor.analyze(conversation)
    │
    ▼
Claude API (提取摘要)
    │
    ▼
MemoryManager.addMemory()
    │
    ▼
StorageService.saveMemory()
```

## 配置加载数据流

```
应用启动
    │
    ▼
ConfigManager.load()
    │
    ├── 读取本地配置文件
    └── 应用默认值
```
