# UI 设计规范

## 设计原则

- 简洁直观
- 响应式布局
- 深色/浅色主题支持

## 色彩规范

| 名称 | 颜色值 | 用途 |
|------|--------|------|
| Primary |  | 主色调 |
| Background |  | 背景色 |
| Surface |  | 卡片/面板 |
| Text Primary |  | 主要文字 |

## 字体规范

| 场景 | 字体 | 大小 | 粗细 |
|------|------|------|------|
| 标题 |  |  |  |
| 正文 |  |  |  |
| 代码 |  |  |  |

## 组件规范

### 消息气泡

- 用户消息：右对齐
- AI 消息：左对齐，支持 Markdown 渲染

### 输入框

- 多行输入
- Shift+Enter 换行，Enter 发送

## 交互规范

- 加载状态：流式打字效果
- 错误状态：内联错误提示
- 空状态：引导提示
