## 数据模型

### 会话表 (sessions)

| 字段 | 类型 | 说明 |
|------|------|------|
| id   | TEXT | 主键 |
| title | TEXT | 会话标题 |
| created_at | INTEGER | 创建时间戳 |
| updated_at | INTEGER | 更新时间戳 |

### 消息表 (messages)

| 字段 | 类型 | 说明 |
|------|------|------|
| id   | TEXT | 主键 |
| session_id | TEXT | 外键 |
| role | TEXT | user / assistant |
| content | TEXT | 消息内容 |
| created_at | INTEGER | 时间戳 |

### 记忆表 (memories)

| 字段 | 类型 | 说明 |
|------|------|------|
| id   | TEXT | 主键 |
| content | TEXT | 记忆内容 |
| tags | TEXT | JSON 标签数组 |
| created_at | INTEGER | 时间戳 |

## ER 图

```
sessions ||--o{ messages : "contains"
memories }o--o{ sessions  : "linked"
```
