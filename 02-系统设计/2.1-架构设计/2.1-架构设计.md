## 模块功能确定

| 模块       | 功能                                                  |
| -------- | --------------------------------------------------- |
| UI模块     | 交互（信息显示与接收）                                         |
| 应用协调模块   | 定义交互接口；协调其他模块的工作流程；                                 |
| 持久化模块    | 保存其他模块需要持久化的数据；保证持久化数据不分散（方便同步）；防止各个模块重复实现相同的持久化功能； |
| 学习场景编排模块 | 抽象调用一个或多个ai的接口；对内保持接口稳定；                            |
| 数据同步模块   | 抽象使用一个或多个间隔重复算法与记忆数据库；对内保持接口稳定；                     |
