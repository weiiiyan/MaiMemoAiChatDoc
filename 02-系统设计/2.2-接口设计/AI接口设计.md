# AI 接口设计

## 接口概述

> 本文档描述与 Claude API（Anthropic）的集成接口设计。

## 请求结构

```json
{
  "model": "claude-sonnet-4-6",
  "max_tokens": 4096,
  "system": "...",
  "messages": [
    { "role": "user", "content": "..." }
  ]
}
```

## 响应处理

- **流式响应**：SSE 事件处理
- **错误重试**：退避策略
- **超时处理**：

## 错误码映射

| HTTP 状态 | 错误类型 | 处理方式 |
|-----------|----------|----------|
| 429 | 限流 | 指数退避重试 |
| 500 | 服务错误 | 提示用户重试 |
