# 测试计划

## 测试策略

| 测试类型 | 覆盖目标 | 工具 |
|----------|----------|------|
| 单元测试 | 核心业务逻辑 | Qt Test |
| 集成测试 | 模块间交互 | Qt Test |
| UI 测试 | 关键用户流程 | 手动 |
| 性能测试 | 响应时间、内存 | 手动 + 工具 |

## 测试范围

### 必测模块

- [ ] ChatManager：消息发送/接收逻辑
- [ ] MemoryManager：增删改查
- [ ] StorageService：数据持久化
- [ ] ClaudeApiClient：请求构建与错误处理

### 非测试范围

- UI 渲染细节
- 第三方库内部逻辑

## 测试环境

| 环境 | 说明 |
|------|------|
| 开发环境 | 本地，使用 Mock API |
| 集成环境 | 连接真实 Claude API |

## 进入/退出标准

**进入**：功能开发完成，代码审查通过。
**退出**：所有用例通过，无 P0/P1 缺陷。
