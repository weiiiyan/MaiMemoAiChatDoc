# 部署与运维

## 构建说明

### 依赖

| 依赖 | 版本 | 安装方式 |
|------|------|----------|
| Qt   | 6.x  | 官网安装 |
| CMake |      |          |

### 构建步骤

```bash
mkdir build && cd build
cmake .. -DCMAKE_BUILD_TYPE=Release
cmake --build . --config Release
```

## 安装包制作

- **Windows**：使用 `windeployqt` + NSIS/Inno Setup
- **macOS**：使用 `macdeployqt` + dmg

## 配置文件

| 文件 | 位置 | 说明 |
|------|------|------|
| config.json | `%APPDATA%/MaiMemoAiChat/` | 用户配置 |
| data.db | `%APPDATA%/MaiMemoAiChat/` | 本地数据库 |

## 更新策略

- 手动下载安装包更新（v1.0）
- 自动更新（后续版本规划）

## 故障排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| 无法连接 API | 网络/Key 错误 | 检查网络和 API Key 配置 |
| 启动崩溃 | 数据库损坏 | 删除 data.db 重启 |

## 日志

- 日志位置：`%APPDATA%/MaiMemoAiChat/logs/`
- 日志级别：DEBUG / INFO / WARN / ERROR
